<h3>Browser hotkeys</h3>

<h4>General hotkeys:</h4>
    <ul>
      <li><b>Alt-R</b> - Refresh Task/View list</li>
      <li><b>Alt-A</b> - Display Attribute List panel</li>
      <li><b>Alt-E</b> - Display Entity List panel</li>
      <li><b>Alt-V</b> - Display Tasks/Views panel</li>
      <li><b>Alt-W</b> - Display Twins panel</li>
      <li><b>Alt-O</b> - Display OE panel (jconsole plugin only)</li>
    </ul>

<h4>After clicking on an entity the following keys are active:</h4>
    <ul>
      <li><b>Home</b> - Set cursor first entity</li>
      <li><b>Page Up</b> - Set cursor prev entity </li>
      <li><b>Page Down</b> - Set cursor next entity</li>
      <li><b>End</b> - Set cursor last entity</li>
      <li><b>Up</b> - Change selected entity to parent</li>
      <li><b>Down</b> - Change selected entity to next hier</li>
      <li><b>Left</b> - Change selected entity to previous sibling</li>
      <li><b>Right</b> - Change selected entity to next sibling</li>
      <li><b>Ctrl-Up</b> - Move subobject child to root (i.e. setToSuboject)</li>
      <li><b>Ctrl-Down</b> - Move subobject root down (i.e. resetSubobjectToParent)</li>
    </ul>

<h4>Changing zoom and position:</h4>
    <ul>
      <li><b>Scroll Wheel Up/Down</b> - Scroll Up/Down</li>
      <li><b>Scroll Wheel Up/Down while ALT pressed</b> - Scroll Left/Right</li>
      <li><b>Scroll Wheel Up/Down while left mouse button pressed</b> - Scroll Left/Right</li>
      <li><b>Scroll Wheel Up/Down while CTRL pressed</b> - Zoom In/Out</li>
      <li><b>Double-click</b> - Toggle between normal zoom and zoomed all the way out</li>
      <li><b>Click hold+drag</b> - Move view port</li>
    </ul>

<h4>Changing attributes displayed in entities:</h4>
    <blockquote>
    By default the browser displays the value of the ID inside the entity.<br/>
    To change this, right-click on an attribute in the Attribute List panel,<br/>
    then select "Attribute displayed".  Multiple attributes may be displayed<br/>
    and they will be listed in the order they were added.  To change ordering,<br/>
    remove the top attributes and then re-add them.  These settings will be<br/>
    saved between sessions.
    </blockquote>
